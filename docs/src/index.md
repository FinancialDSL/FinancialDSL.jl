
# FinancialDSL.jl

[DSL](https://en.wikipedia.org/wiki/Domain-specific_language) for Financial Contracts.

## Requirements

* Julia v1.2 or newer.

## Installation

From a Julia session, run:

```julia
julia> using Pkg

julia> Pkg.add("FinancialDSL")
```

## Source Code

The source code for this package is hosted at
